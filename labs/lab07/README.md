# Lab 7: Linear SVM & PR-Logic (UA News Multiclass)

1. **Підзадача:** Багатокласова класифікація українських новин (Політика, Бізнес, Спорт, Технології, Новини).
2. **Baseline ЛР6:** Logistic Regression + TF-IDF word(1,2).
3. **SVM-варіанти:** Протестовано `LinearSVC` зі словами та комбінований варіант із char-ngrams (3,5). Останній виявився найкращим.
4. **Дисбаланс:** Присутній (Політика - 33.5%, Технології - 9.7%). Використано `class_weight='balanced'`.
5. **PR-Логіка (One-vs-Rest):** Для аналізу виділено клас `технології`. Зміщено поріг до `-0.4` під логіку *Recall-first* (максимізація повноти виявлення ІТ-новин для агрегатора).
6. **Найкраща модель:** `LinearSVC (word+char, balanced)`.
