# Лабораторна робота 6: TF-IDF + Logistic baseline (UA News)

**1. Напрям проєкту:** Напрям **А (Text Classification)**.

**2. Classification-підзадача:** Багатокласова класифікація текстів українських новин за 5 рубриками (політика, спорт, новини, бізнес, технології).

**3. Порівняння baseline-варіантів:**
* **Baseline 1:** `TfidfVectorizer(1,2)` + `LogisticRegression(max_iter=1000)`
* **Baseline 2:** `TfidfVectorizer(1,2)` + `LogisticRegression(class_weight='balanced')`

**4. Основні цифри (на Test Set):**
* **Baseline 1 (Найкращий):** Accuracy = 0.9187 | Macro-F1 = 0.9131
* **Baseline 2:** Accuracy = 0.9145 | Macro-F1 = 0.9100
*Балансування класів не дало приросту, оскільки навіть міноритарний клас має достатню вибірку (>11k текстів).*

**5. Результати Error Analysis та наступні кроки:**
* **Витік даних (Leakage):** Аналіз Top Features показав, що модель "чітерить", використовуючи назви інформагентств ("еспресо", "нв бізнес") як індикатори класів. 
* **Перетин класів (Overlap):** Матриця помилок показала найбільшу плутанину між рубриками "Політика", "Бізнес" та загальними "Новинами".
* **Що фіксити далі:** Необхідно розширити стоп-словник іменами медіа та журналістів, щоб змусити модель вчити справжню семантику новин, а також спробувати ембединги замість TF-IDF для кращого розпізнавання контексту.
